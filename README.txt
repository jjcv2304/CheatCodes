
The application allows the user to enter and search information.

The goal of the project is to practice with different development tools and concepts.


***************
*** Backend ***
***************

List of frameworks/libraries used: identityserver4, Swagger, Serilog, Dapper(NHibernate/EF in earlier commits), 
CQRS, XUnit, F#, Patterns(UOW, Repository, Builder,..), DDD, Layers(Api, Domain, Persistance,..), RabbitMq, 
HealthChecks, Https, Extension points(SecurityHeadersMiddleware, Global error catch), Docker(branch Docker_support)


***************
*** Frontend ***
***************

List of frameworks/libraries used: oidc-client, Angular Material, NgRx, UnitTest, marble testing...
